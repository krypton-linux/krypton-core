# krypton-core

Krypton Linuxのpacmanパッケージリポジトリ。

# 利用可能なサーバー

メインサーバー(推奨)

```pacman.conf
[krypton-core]
Server=https://krypton-linux.com/krypton-core/repo
SigLevel=Never
```

pik6c(Kaedehito)様のミラーサーバー

```pacman.conf
[krypton-core]
Server=https://krypton-linux.pik6c.tech/downloads
SigLevel=Never
```
